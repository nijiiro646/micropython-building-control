# Micropython Building Control

(Work in progress) A simple proof-of-concept for an integrated building control system using micropython and a Raspberry Pi Pico W microcontoller. Created as part of an introductory-level design project.

Intended as a minimum viable product, it implements a web-based UI with two-way communication and other basic server functionality. There is also a user authentication system to prevent unauthorized users from changing the settings.
